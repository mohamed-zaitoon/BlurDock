# BlurDock

BlurDock is a system-level module designed for Xiaomi devices running HyperOS.

## ✨ Features
- Install BlurDock as a system app
- Optimized for Xiaomi HyperOS
- Lightweight and fast
- Compatible with Android 15+

## 📱 Requirements
- Xiaomi device only
- HyperOS 2 or higher
- Android 15+

## ⚙️ Installation
1. Install via Magisk or KernelSU
2. Reboot your device
3. Enjoy BlurDock
